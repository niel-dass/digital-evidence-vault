About

The Devault browser application allows you to secure digital evidence during a crime investigation. It does this by storing the hash of a file on the ethereum blockchain through a smart contract. Once the hash has been saved, it cannot be altered. This is because the decentralized nature of blockchain technology stores the data on thousands of computers around the world, which are all processing transaction blocks using crytography. Such a system creates an effective way to detect evidence tampering since an intruder who attempts to change a file cannot change the original hash without an immense amount of computing power.


Instructions

  1. To create a case, go to the "Submit case evidence" section.
  2. Enter the case ID.
  3. Upload the file containing all the evidence for that case.
  4. Make sure that all the data you entered is correct because you can submit a file only once for that ID.
  5. When you are ready to proceed, click "Submit"
  6. To verify a file go to the "Verify integrity of evidence" section.
  7. Enter the ID of the case you are investigating.
  8. Upload the file containing the evidence for that case.
  9. Click "Verify"
  10. A messaage will pop up at the bottom indicating whether the file has been tampered with.
